How to Apply These Styles

The OctoFlow theme is already loaded in your app!

To see these styles in your actual OctoFlow application, you need to add the CSS classes to your components:

  • Add octoflow-card to card elements
  • Add octoflow-btn-primary to primary buttons
  • Add octoflow-btn-secondary to secondary buttons
  • Add octoflow-input to input fields
  • Add octoflow-badge to badges/tags

The theme CSS is in: /web/app/styles/octoflow-theme.css
